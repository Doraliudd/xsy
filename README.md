沙盒环境与生产环境的物理隔离：

• 沙盒环境与生产环境在物理服务上的隔离：独立的代码运行服务器；独立的数据库服务；独立的网络安全防护等

• 沙盒环境中的文件相关的内容存储也需要独立的环境来进行文件存储服务

• 沙盒环境的运行不会影响生产环境的正常运行
